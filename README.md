# 📊 Smart Attendance Pattern Analyzer

A data-driven **Machine Learning and Data Analytics** project that analyzes student attendance behavior, uncovers hidden patterns, and provides actionable insights through statistical analysis and interactive visualizations.

By simulating academic schedules, exam periods, holidays, and student behavior, the system identifies attendance trends, bunking patterns, and contextual factors that influence classroom participation.

## ✨ Features

### 📈 Attendance Analytics

* Analyze attendance trends over time
* Calculate attendance percentages
* Detect absenteeism patterns
* Compare attendance across different courses

### 🎓 Academic Context Analysis

* Study attendance before and after examinations
* Analyze holiday and vacation impact
* Evaluate weekday-wise attendance behavior
* Understand semester timeline trends

### 👥 Student Behavior Analysis

* Identify frequent bunking patterns
* Analyze co-absence relationships
* Discover attendance clusters
* Detect group attendance behavior

### 📊 Statistical Analysis

* Chi-Square Test
* ANOVA Analysis
* Descriptive Statistics
* Correlation Analysis

### 📉 Interactive Visualizations

* Attendance Heatmaps
* Timeline Analysis
* Distribution Graphs
* Network Graphs
* Comparative Charts

## 📌 Project Workflow

1. Generate or load attendance dataset.
2. Clean and preprocess the data.
3. Perform exploratory data analysis (EDA).
4. Analyze attendance across different academic contexts.
5. Build attendance relationship networks.
6. Perform statistical validation.
7. Generate visualizations and insights.

---

## 📊 Key Insights

* Attendance varies significantly during examination periods.
* Holidays influence attendance behavior before and after scheduled breaks.
* Distinct weekday attendance trends emerge over the semester.
* Student co-absence patterns reveal social attendance behavior.
* Statistical testing validates significant attendance differences across academic contexts.

## ⚙️ Installation


